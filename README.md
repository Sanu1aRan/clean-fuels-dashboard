# 🌍 Clean Fuels for Cooking Dashboard

An interactive Streamlit dashboard analysing global access to clean cooking fuels and technologies (SDG 7.1.2).

## Live App
https://clean-fuels-dashboard-ejrvgrf7rnhr8tc2r9hkc2.streamlit.app/

## Dataset
World Bank Indicator: EG.CFT.ACCS.ZS  
Source: IEA / WHO / World Bank — Tracking SDG 7: The Energy Progress Report (2023)

## Features
- 🗺️ Interactive world map by year
- 📈 Regional trends (2000–2023)
- 🔍 Country deep-dive with urban/rural breakdown
- 🏙️ Urban vs Rural access gap analysis

## How to Run Locally
```bash
pip install streamlit pandas plotly
streamlit run app.py
